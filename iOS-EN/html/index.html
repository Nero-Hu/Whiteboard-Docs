Agora

  • Hierarchy

Agora provides interactive whiteboard with high reliability and rich features though a virtual global network and flexible combinations of APIs.

  • WhiteSDK class provides the main methods for initializing the WhiteSdk object, joining the live Interactive Whiteboard room, and create a WhitePlayer object.
  • WhiteRoom class provides methods that manage the live Interactive Whiteboard room.
  • WhitePlayer class provides methods that manage the playback of the whiteboard content.
  • WhiteDisplayer is the parent class of the WhitePlayer class and the WhiteRoom class. The WhitePlayer and WhiteRoom classes can both inherit the methods of the WhiteDisplayer class.
